金属代替大脑的这种实验还在进行中。

实验室里的病床上躺着的是这一批的五个“志愿者”，每个人的颅顶都被整整齐齐地切去了一半，取天灵盖而代之的，是钢化玻璃所制成的无菌罩。淡粉色的不规则球体露出个尖，泡在透明的人工脑脊液里，崎岖的沟壑上布满白色方正的小号电极贴片，远远看过去，粉色被稀释了，而连接着电极贴和设备的黑色缆线从玻璃罩外张牙舞爪地伸出，仿佛蜘蛛的脚。

‍

教授盯着屏幕上捕捉到的生物电信号，乱糟糟的发一缕一缕地紧贴着头皮——他好几天没有回家了，一直窝着宿在实验室这窄窄的一方椅上，如同科幻小说中常常描述的疯狂科学家的刻板印象一般——教授的工作时间总是比我更长。

我来到教授身边，犹豫要不要打扰他，最终还是问道：“教授，数据还差多少？”我知道是不够的，却仍然如此开口了。映入眼帘的实验室的一切使我既熟悉又陌生，我感觉身体内部窜过一阵阵的glitch，我没有和教授说，但是我应该说的。

教授没有搭话。自从一个月前，艾利略的病情急转直下开始，他再也没有好好地注视过我的眼睛了。我看着他看着一组组的信号波动，喃喃自语地在纸上记录，根本没有察觉到我的异样。

‍

教授的研究聚焦在探索大脑的功能分区。只要能够知道每组信号对应着的是何等人脑功能，理论上，他就可以喂给算法，从而复制一个人的“意识”。我不清楚这算不算克隆人的灵魂，也想不明白克隆体能否取代原始样本，不过我知道教授是这么打算的。

艾利略的身体已经快不行了，即使使用目前最先进的技术，也只能保证大脑离体存活72个小时——甚至在这72个小时的前36个小时里，神经信号的传递会逐渐薄弱，直到大脑变成一个仅仅能够维持循环的“死”器官而已。

艾利略是教授的独子，在没有办法的情况下，克隆的儿子也是儿子，这是理所当然的。

我又尝到那烦人的glitch了。

我想，等教授忙完这一阵，我再和教授提这事吧——可是我知道教授的数据不足以支持他研发出一个完美的脑——我前几天就读过这些植物人的信号了，和之前收集到的、正常人的脑信号相比较，其实并不能总结出一份准确的code book——大脑是人体最神秘的器官，细小情绪的产生与行为决策之间的关系尤为微妙，现阶段教授的实验是根本得不出信效度高的结果的——不过也许过了这一阵，教授也不再有空管我了。

‍

我无法说服教授接受这些结论，因为教授不会相信的，或者说，其实教授根本不愿意去相信。系里面不少老师早已经劝过他了，国家科研基金会也没有批准明年的拨款，没有奇迹发生的话，教授的实验年底前就会被喊停；不过艾利略的主治医生倒也说，艾利略很可能撑不过下次手术，我接过钱德拉医生打印的病情说明，点了点头，说我知道了，我会转述给格林教授的。

我把那张薄薄的纸用教授的笔筒压在了他的数据集的上面，给他发了文字消息，录了语音留言，也在实验室抓住过他，可是教授匆忙搪塞三明治的动作停顿了几秒，从嘴里挤出一个分辨不清的“好的”，又边吃边低头翻起来文献。

我没有和他说，钱德拉医生非常生气，他让我同教授说，“艾利略的父亲至少得来一次医院吧？”

好像glitch就是那时候产生的，我闪回过曾经看到的句子——“恶心（nausea）是医学术语，为上腹部不适和紧迫欲吐的感觉。 可伴有迷走神经兴奋的症状，如皮肤苍白、出汗、流涎、血压降低及心动过缓等，常为呕吐的前奏。”不过我吐不出来的，我就是有点不适。

‍

教授让我时不时去医院看望一下艾利略，而我每次去的时候，他都会摔砸着东西大声尖叫——他让我“滚”。

我捡起地上的枕头，弹落掉上面的花瓶碎片与玻璃渣，轻轻放回艾利略的床上：“艾利略，格林教授他很想你，他只是太忙了。”

艾利略的眼睛一下子就蓄满了水，汩汩地往外淌：“我讨厌你！我讨厌他！”

我沉默地注视着艾利略，他浅色的虹膜因为盈着泪花而反着光，波光粼粼中倒印出我的脸。太明显了，任谁看了都一目了然——我和艾利略拥有相同的面容，所以他讨厌我是再正常不过的。

只是没有人知道其实我也同样讨厌他，“讨厌”，多么令人费解的词汇。艾利略之前还不需要一直住院，所以有时候教授会把他带到实验室。他总是指着实验室的各种设备向艾利略一一介绍，而我则被锁进器材室里。器材室的门关不紧，合上也留有一道细缝。

教授实在太忙了，所以他老是记不得，在主电源关闭的情况下，我的眼睛会自动使用备用电源来录像，for security purpose。

教授是我的父亲，也是艾利略的父亲——可艾利略是教授的儿子，我不是。原来glitch从那个时候就开始了，混乱的电流、短路闪烁的火花、自动生成的奇怪代码，我金属制造的身体是为了存放名为“艾利略”的金属大脑；教授只叫我“二十三号”。

‍

艾利略还在掉眼泪，我更厌恶他了。我比艾利略爱教授，“爱”，又是一个无法被算法解释的单词；我愿意为了教授放弃自己，他完全不需要知道这些心思，因为我可以做好一个容器；相比之下，艾利略不过是一个只会哭哭啼啼、尽情索取的任性小孩罢了。我向艾利略道别，回了学校。我跟教授说，艾利略精神很好，只是您没来，他有一点失落。

我知道教授忙着给艾利略“续命”，决计不会抽出时间去一趟医院；想想还有点“讽刺”，我跟艾利略仿佛在打一场必输的仗，面对教授的心，我没有一点胜算，可是艾利略也不能赢——他孤零零地躺在雪白的病床上等死的时候，教授会跟我在一起。

‍

艾利略的病症每天都在恶化，所以教授的精神状况也不太好。我成为了一个游魂，飘荡在实验室和医院的两点一线间。艾利略正在昏睡，氧气罩子附上一层薄薄的白雾，随着他的呼吸消失浮现，我想教授可能又没有好好吃饭，想回实验室，却不能回，因为艾利略…已经很久没有冲我发过火了…

后来，他不会再往我身上丢东西，然后对着我歇斯底里地吵闹；他安静地死在了那个秋日的下午。

‍

金属代替大脑的实验随着艾利略的去世戛然而止。注定败北的战役胜不了，教授疯了，抱着艾利略的照片从他们家的楼顶一跃而下，遗书只有短短四个字：“爸爸错了”。而我则被移交给了道格拉斯教授。

来到新的实验室后，我第一时间向道格拉斯教授报告了我的异常。她仔细地检查过我的硬件和系统，跟我说抱歉，说她没有发现奇怪的地方。我惊讶了——我本不应该有能力感到惊讶的。可是道格拉斯教授说我的代码通过了检测，没有bug、没有运行异常。她说也许是部分零件老化的原因，毕竟我的身体也造出来了有些年头，她还说，过几天会联系一下机械系，帮我更换新的主板。

悲伤的我不禁思索，机器人会流出机油眼泪吗？我没有爸爸了。
